# Project: AIF (Army In Fantasy)

# 게임명: No Mana Just Gun !!

# 개요
본 프로젝트는 1인칭 슈팅 디펜스 게임 입니다.
몬스터를 처치시 얻는 경험치를 통해 레벨업을 합니다.
레벨업을 할 때 얻는 능력치를 통해 성장하여 
10분동안 크리스탈을 지켜내도록 합시다.
(크리스탈에게는 숨겨진 기능이 있습니다)

- 장르: 1인칭 슈팅 디펜스
- 플레이 타임: 약 10분
- 개발환경: 2022.3.62.f2
- 플랫폼: Windwos
- 프로젝트 기간: 2026.01.28 ~ 2026.02.03

# 게임 특징
- 무기 교체 기능
- 수류탄 투척
- 다양한 몬스터
- 크리스탈 스킬
- 풀 더빙

# 조작법
- 이동: WASD
- 달리기: Left Shift (누르는 동안)
- 점프: 스페이스
- 공격: Mouse Left Click
- 크리스탈 상호작용: F
- 게임 멈춤: ESC
